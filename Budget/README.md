# 💰 UGX Budget Tracker

A simple command-line budget tracker built in Python for tracking income and expenses in Ugandan Shillings (UGX).

Built with pure Python — no external libraries needed.

---

## Features

- Add income and expenses with descriptions
- View a full budget summary with totals and balance
- See all transactions in a clean list
- Set a savings goal and track your progress with a visual bar
- Clear all transactions and start fresh

---

## How To Run

Make sure you have Python 3 installed. Then:

```bash
python budget_tracker.py
```
